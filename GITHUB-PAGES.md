# GitHub Pages Configuration

Para configurar GitHub Pages:

1. Ir a Settings del repositorio
2. Seleccionar "Pages" en el menú lateral
3. En "Branch", seleccionar "main" y carpeta "/ (root)"
4. Guardar los cambios
5. La aplicación estará disponible en: https://[usuario].github.io/App-Contador/

## Estructura recomendada:

- Archivo principal: index.html en la raíz
- No se necesitan carpetas adicionales
- GitHub Pages servirá automáticamente el archivo index.html
